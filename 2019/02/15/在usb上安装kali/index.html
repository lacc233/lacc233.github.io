持久加密Usb安装（基于Live）

  • 技术依赖
    LUKS：linux Unified Key Setup 磁盘分区加密规范（文件被放在加密的分区内）
    不依赖于操作系统的磁盘级加密
    后端：dm-crypt（组件）
    前端：cryptsetup（命令行工具）
    Windows：DoxBox（组件），可以查看LUKS加密的分区
    微软的bitlocker，磁盘级加密
